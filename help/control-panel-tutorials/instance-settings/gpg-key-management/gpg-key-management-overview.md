# GPG 密钥管理 - 概述

GPG (GnuPG) 使用非对称的单个密钥对加密消息。要使用 [!DNL Campaign] 实施 GPG 加密，管理员必须直接从控制面板在营销实例上安装和/或生成 GPG 密钥。

了解如何在指定的 [!DNL Campaign] 实例上生成和安装公钥/私钥对。
