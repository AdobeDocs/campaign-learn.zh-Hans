# 控制面板 - 概述

通过控制面板，Adobe Campaign 管理员可以监测关键资产并执行管理任务，例如按实例管理 SFTP 存储、将 IP 地址列入允许列表或管理 GPG 密钥。

请参阅 [发行说明](https://experienceleague.adobe.com/docs/control-panel/using/release-notes.html?lang=zh-Hans) 有关最新产品版本的信息。
