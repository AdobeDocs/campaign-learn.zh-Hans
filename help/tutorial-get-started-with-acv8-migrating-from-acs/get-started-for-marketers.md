# 营销人员入门

本指南概述了Campaign v8的主要功能。 它适用于从Campaign Standard迁移到Campaign v8的营销人员。

您可以通过客户端控制台或Web用户界面访问Adobe Campaign v8。 通过Web界面，您可以创建、管理和执行关键营销操作。

>[!NOTE]
> Adobe Campaign Web用户界面版本在持续交付模型上运行，通过该模型可采用更具可扩展性、分阶段的方法部署功能。 请定期查看[发行说明](https://experienceleague.adobe.com/en/docs/campaign-web/v8/release-notes/release-notes)以了解最新更新。
