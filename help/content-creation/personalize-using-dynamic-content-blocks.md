# 使用动态内容块个性化电子邮件

在 Adobe Campaign 中，您可以个性化内容或消息的外观，并将它们与收件人用户档案的条件相结合。

个性化块是动态的、个性化的，并包含可添加到投放中的特定渲染代码。例如，您可以添加图像、电子邮件页眉、电子邮件页脚、镜像页面链接、退订链接等。
