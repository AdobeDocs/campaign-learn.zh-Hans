# 配置投放模板

营销部门制作投放模板，以规范创意外观，能够更快地执行和启动活动。

模板可以系统地包括：

* 类型
* 发件人和回复地址
* 基本的个性化块
* 镜像页面和退订链接的链接
* 内容、公司徽标或签名
* 其他投放属性，如资源有效性、批次数量…
